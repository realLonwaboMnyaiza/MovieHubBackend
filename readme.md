# Movie Hub ASP.NET Project

Backend ASP.NET API application used for viewing all the movies we have to offer.

## Tech Stack

- C#
- Entity Framework
- MySql

## Getting Started

- Install Dependencies:
  `dotnet restore`
- Build:
  `dotnet build`

---

- Run App Locally `dotnet run`
- Use REST Client i.e. Postman to query movies.
